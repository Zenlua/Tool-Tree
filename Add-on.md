
Định dạng tệp add-on:

- Là một file nén dạng .zip hoặc .7z

- Sau khi nén xong, đổi tên đuôi thành file.add

Cấu trúc bên trong tệp add-on:

```
file.add
└── (nội dung bên trong tệp)
    ├── addon.prop
    ├── icon.png (256×256)
    ├── menu.sh|menu.xml
    ├── index.sh|index.xml
    └── addon/
```
