## ✨ Features

**📦 Unpack**

`boot`, `dtbo`, `ext4`, `erofs`, `f2fs`, `payload`, `super`
`zip`, `apk`, `apks`, `apex`, `capex`, ...

**📥 Pack**

`boot`, `dtbo`, `ext4`, `erofs`, `f2fs`
`payload`, `super`, `amlogic`
`apk`, `apex`, `capex`, ...

> And many more tools waiting to be discovered.

## 📂 Working Directories

### Input

+ `/sdcard/TREE`
- ROM
- APK

### Root Working Directory

APK: `/data/local/TOOL`

ROM: `/data/local/TREE`
